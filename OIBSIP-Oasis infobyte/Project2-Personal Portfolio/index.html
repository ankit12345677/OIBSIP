Online Sports Center

I developed an "Online Sports Center" web application utilizing React.js for the frontend and Node.js for the backend. The platform features robust user authentication and includes a specialized seller dashboard for the systematic categorization of sports products. Administrators are empowered to categorize products based on various sports, exposing the catalog through RESTful APIs for customer viewing. The application seamlessly facilitates customer product selection with automated price calculations and offers flexible payment options, ensuring a secure transaction process. This comprehensive system ensures a dynamic and user-friendly experience, combining an intuitive interface with efficient backend processes to provide customers with a reliable and secure means to select and purchase sports products for prompt delivery.
